Swivori Privacy Policy

Swivori is a lightweight creator tool that helps users build image-generation prompts. This page explains how the app handles information.

Effective date: June 10, 2026

1. Who we are

Swivori is provided by Viori Studio. If you have any questions about this Privacy Policy, you can contact us at privacy@swivhome.com.

2. What Swivori does

Swivori helps users turn short scene descriptions into structured image-generation prompts. Users can write scene ideas, choose a style, and generate prompt text inside the app.

3. Prompt generation and user input

Swivori does not send the user’s scene descriptions or generated prompts to Viori Studio servers.

Prompt generation is performed locally on the user’s device.

User-written scene descriptions and generated prompts are not collected, shared, sold, or transmitted by Viori Studio.

4. Local storage

The app may store user-written scene descriptions, style selections, generated prompts, recent items, saved items, or app settings locally on the user’s device to support normal app usage.

This local data stays on the user’s device unless the user chooses to copy, export, share, or otherwise move it outside the app.

This local data is not stored on a Viori Studio server.

5. User sharing and third-party image tools

Swivori creates prompt text that users can copy and use in external image-generation tools or other apps.

If a user chooses to paste, submit, share, or upload a prompt outside Swivori, that action is controlled by the user and may be handled under the privacy policy of the external service the user chooses to use.

Viori Studio does not control how third-party apps, websites, or image-generation services process information that users choose to provide to them outside Swivori.

6. Advertising and Advertising ID

Swivori does not display ads.

Swivori does not use the Google Mobile Ads SDK.

Swivori does not collect, access, or use the Android Advertising ID.

Swivori does not use advertising networks or personalized advertising.

7. Accounts and server-side user data

Swivori does not require users to create an account.

Swivori does not provide login, account registration, or account-based cloud storage.

Because Viori Studio does not operate user accounts or store user prompt data on its own server, there is no server-side account data for us to delete upon request.

Users can delete locally stored app data by clearing the app’s data or uninstalling the app from their device.

8. Analytics and diagnostics

Swivori does not use analytics SDKs to track user behavior.

Swivori does not use crash reporting SDKs that transmit diagnostic data to Viori Studio or third-party analytics services.

9. Children’s privacy

Swivori is not designed to knowingly collect personal information from children through Viori Studio servers.

Because Swivori does not operate user accounts or collect user prompt data on Viori Studio servers, Viori Studio does not knowingly collect personal information from children through the app.

10. Changes to this Privacy Policy

We may update this Privacy Policy from time to time. If we make changes, we will update the effective date on this page.
